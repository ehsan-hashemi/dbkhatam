چطور نام کاربر را در پروفایل به او نشان دهیم؟

سلام! در این قسمت از مجله گروه برنامه نویسی احسان، می‌خواهیم آموزش دهیم که چطور نام کاربر را در پروفایل به او نشان دهیم. یعنی وقتی کاربر وارد حساب کاربری خود می‌شود،‌ در پروفایل به او خوش آمد گویی کنیم. البته این روش به درد کار های دیگر هم می‌خورد. مثلا می‌خواهم به جای اینکه کلی در سایت دنبال این بگردیم که قیمت این محصول یا این متن رو چی گذاشتیم، میتوانیم آن را در یک متغیر جاوا اسکریپت در فایلی ذخیره کنیم تا دفعه‌ی بعد به راحتی او را ویرایش دهیم.

کد HTML

ابتدا یک فایل html ایجاد می‌کنیم و این کد را در فایل ذخیره می‌کنیم.

Html 
<h1>پروفایل</h1>
<p>کاربر گرامی <span id="username"></span> خوش آمدید.</p>

در این کد داخل تگ p یا هر تگ دیگری که می‌خواهید در آن به کاربر خوش آمد بگویید، یک تگ span بگذارید و داخل آن هیچ چیزی نگذارید. فقط یک id برای او بذارید و داخلش username بگذارید برای ارتباط با ‌جاوا اسکریپت.

کد Java Script

سپس این کد جاوا اسکریپت را در پروژه می‌گذاریم:

Java Script 
function updateMessage() {
    const now = "hi";
    document.getElementById("username").textContent = "احسان هاشمی";
}

updateMessage();

حالا برای تغییر نام کاربر یا متنی که می‌خواهید تغییر دهید، باید به جای "" در عنصر username متنی که می‌خوهید را وارد کنید.

خروجی

خروجی این کد با کمی css به صورت زیر است:

پروفایل

کاربر گرامی خوش امدید.

امیدوارم روزی این آموزش به کارتان بیاید. ما را دنبال کنید و از دیگر مطالب بهره مند شوید.
